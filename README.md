# ❤️ Motfrais Kiss

> Un cóctel de autor convertido en experiencia web.

---

## ✨ Descripción

**Motfrais Kiss** no es solo un cóctel…  
es un homenaje.

Un proyecto que combina:
- 🍸 Coctelería de autor  
- 🎨 Diseño UI moderno  
- 🎧 Música original  
- 💖 Historia personal  

Todo en una sola experiencia interactiva.

---

## 🌸 Inspiración

Inspirado en **Motfrais**, una streamer que transformó noches comunes en momentos especiales.

El cóctel busca representar:

- Dulzura 🍬  
- Calidez ☀️  
- Compañía 🤝  
- Identidad 💫  

---

## 🍸 El Cóctel

**Motfrais Kiss**

Un cóctel suave, dulce y reconfortante, diseñado para:

> "quedarte un rato más"
